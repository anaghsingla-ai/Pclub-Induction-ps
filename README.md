# 📚 Study Room Booking UI

A modern and responsive **Study Room Booking Interface** built using **HTML, CSS, and JavaScript**. This project simulates a real-world booking system with an elegant UI and smooth user interactions.

---

## 🚀 Features

- 🗓️ Room booking interface  
- 🎯 Interactive UI with modern design  
- 🌙 Dark mode support  
- ⚡ Real-time booking feedback (toast notifications)  
- 📱 Fully responsive layout  
- 🎨 Advanced CSS styling (flexbox, animations, transitions)  

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3 (Advanced Styling + Animations)**
- **Vanilla JavaScript (DOM Manipulation)**
